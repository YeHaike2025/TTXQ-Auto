简介

TTXQ-Auto 是一套全自动天天象棋对弈系统。通过屏幕截图识别棋盘局面，调用 Pikafish 引擎计算最佳走法， 再通过模拟鼠标点击执行落子，实现完整的“截图 → 识盘 → 思考 → 走子”闭环。 支持自动检测红 / 黑方、5 档棋力、实时 HUD、AI Agent 导航接口、游戏结束自动重启等功能。

⚠  推荐在 Windows 中使用，搭配 CodeBuddy 或 WorkBuddy， 模型选择 Claude-Sonnet-4.6 或 GPT-5.4（不推荐 Claude-Opus，该模型在 CodeBuddy 上无法正常识别图片）。
🔔 不同 AI 工具的使用方式
CodeBuddy
支持全自动方式：直接对话“使用 TTXQ-Auto 这个 Skill 自动下天天象棋”， AI 能识别截图坐标并自动完成启动 → 导航 → 进入对局的全流程。
Tiki 小龙虾
无法从截图中返回按钮的正确坐标，因此需要手动操作： 先手动点击进入“天天象棋小游戏”的象棋对局界面，棋盘出现后，再告诉 Tiki 小龙虾：
使用 TTXQ-Auto 自动下天天象棋
即可接管自动下棋。

快速开始

一、准备天天象棋小游戏环境

下载并安装“腾讯应用宝”

下载地址：https://sj.qq.com/

在腾讯应用宝中搜索并下载“天天象棋 小游戏”

安装完毕后，应用宝桌面会出现“天天象棋 小游戏”的快捷入口。

二、在 AI Agent 中使用本 Skill

安装 Skill

在 AI Agent 中发送如下指令，让其自动安装本 Skill（示例路径请替换为实际路径）：

安装此 Skill，路径为 D:/006_TTXQ-Auto_v1.6.1_20260425.zip

启动自动下棋

安装完成后，对 AI Agent 发送：

使用 TTXQ-Auto 这个 Skill 自动下天天象棋

即可开始全自动对弈流程。

进入自动下棋

本 Skill 支持两种方式进入自动对弈：

📱 手动方式

手动点击进入“天天象棋 小游戏”的象棋对局界面，然后由本 Skill 接管自动下棋。

适用场景：Tiki 小龙虾，或希望自己选择对手难度的情况。

🤖 全自动方式

由本 Skill 自动启动天天象棋游戏，并自动进入“人机对战”。

适用场景：CodeBuddy / WorkBuddy，开箱即用，只需一句话。

功能特性

🆕

完全自主运行

启动后自动发现窗口、截图、识别、下棋、等待对手。

📺

自动启动 APP

未找到窗口时自动通过桌面快捷方式启动天天象棋。

⚖

自动检测红 / 黑方

通过底部帅 / 将位置判断我方颜色。

🔄

对手走子检测

通过 FEN 轮询检测对手是否已走完，替代固定时间等待。

🎯

游戏结束自动重启

对局结束后自动检测并等待新对局开始。

👁

实时 HUD 覆盖层

透明显示步骤信息和走法箭头，按键：F9 HUD · F10 Log · F11 Folder。

💡

操作可视化反馈

截图（金色矩形）和点击（青色十字）在屏幕上短暂显示。

🧠

AI Agent 导航接口

提供截图 + 分析 + 点击 API，让 AI Agent 辅助导航到对局界面。

📊

游戏状态检测

识别 IN_GAME / GAME_OVER / POPUP / MAIN_MENU 等界面。

📋

丰富对局报告

每步详细信息 + 可视化图片 + 统计汇总。

🎯

YOLO11 棋子识别

15 类分类模型，准确率 > 95%。

♚

Pikafish UCI 引擎

中国象棋专用引擎，接近特级大师水平。
